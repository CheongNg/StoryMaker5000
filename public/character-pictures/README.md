# Character Pictures

Drop local character reference pictures in this folder.

Files in this folder are served by the local app at:

```text
/character-pictures/file-name.jpg
```

Example:

```text
public/character-pictures/mira-vale.jpg
```

Use it in the app or notes as:

```text
/character-pictures/mira-vale.jpg
```

## Naming habit

Use lowercase file names with hyphens:

```text
mira-vale.jpg
jonathan-grey.png
scene-reference-01.webp
```

Avoid spaces, special characters, and very large files.

## Recommended formats

- `.jpg` for photos or full illustrations
- `.png` for transparent cutouts or clean references
- `.webp` for compressed browser-friendly images

## Suggested size

Keep each image around `1024px` to `1600px` on the longest side when possible. This is large enough for character reference, but not so large that the local app becomes sluggish.
