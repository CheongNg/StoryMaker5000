# Priority Template PDFs

Put complete, high-quality story PDFs in this folder.

These PDFs should be treated as the strongest examples for StoryMaker5000 to learn from. Use them when the story is already polished and should influence the target quality bar more than ordinary supplemental examples.

## What belongs here

- Complete picture books
- Polished story samples
- Strong examples of pacing, page turns, tone, and illustration flow
- Stories that represent the quality level you want StoryMaker5000 to imitate

## What does not belong here

- Loose inspiration images
- Draft story text
- Partial scenes
- Random screenshots
- Character-only references

Put those in `story-library/stories/.../references/` instead.

## Naming habit

Use lowercase file names with hyphens:

```text
gentle-bedtime-story.pdf
funny-forest-adventure.pdf
friendship-picture-book.pdf
```

## Priority index

When you add a PDF, also add a short entry to `index.md`.
